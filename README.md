# Rehan Wasi — Landscape Architect Portfolio

Portfolio site built with React + TypeScript + Tailwind CSS + Framer Motion.

## Dev

```bash
npm install
npm run dev
```

## Stack

- Vite + React + TypeScript
- Tailwind CSS v3
- Framer Motion
- Lucide React
- Kanit (Google Fonts)
